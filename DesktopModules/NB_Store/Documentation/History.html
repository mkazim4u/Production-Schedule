UPGRADE WARNING!!
This version of NB_Store has removed the double search in settings. In the old version, If a setting was not in the setting section, NB_Store looked into the templates for a template with that name. This function has been removed, this may cause issues with setting/template not displaying. If you have problems with displaying templates then make sure your settings are in the settings section and not the template section of the backoffice.
